# Python-AI-Mastery

A daily log of Python fundamentals, building toward AI/ML engineering topics. Each `dayNN/` folder holds that day's exercises.

## Progress

- **day01** — Hello world, a basic calculator, and an intro to variables/f-strings (`hello.py`, `calculator.py`, `student.py`)
- **day02** — Variables, data types, type conversion, and user input (`variables.py`, `datatype.py`, `type_conversion.py`, `user_input.py`, `user_profile_formatter.py`)
- **day03** — Arithmetic operators, including floor division, modulus, and exponentiation (`arithmetic.py`)

## Running

Each script is standalone:

```bash
python3 day01/hello.py
```

A virtual environment is set up in `.venv/` — activate it with `source .venv/bin/activate` before running scripts if you want an isolated environment.
